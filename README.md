## Build and run locally

### Clone master branch

```
$ git clone -b master https://github.com/girisagar46/girisagar46.github.io.git
```

### Serve HTML

```
    $ cd girisagar46.github.io
    $ python -m SimpleHTTPServer
```
### Browse to `http://localhost:8000`
